## Useful QA Tools

There are several opensource and paid tools for bug tracking and test cases management:
- Testlink
- Bugzilla
- JIRA 
- Taiga
- Clickup
- Trello
- GSuite
- Github (?) ....yes, you can write test cases using markdown and to do bug tracking using tab "Issues"
